Project Journey

A.L.T.R. began as a Grade 11 I-STEM engineering project focused on designing a practical solution to environmental litter collection. After completing the initial project in class, my teammate and I enjoyed working on it so much that we decided to continue developing the idea independently. This eventually led us to submit it to the Tech Under 20 (TU20) entrepreneurship competition.

V1

School Project

The initial version was built during our Grade 11 I-STEM engineering class. This prototype featured a cardboard chassis with a 3D-printed claw mechanism. The control system was based on a micro:bit microcontroller, with DC motors powering the wheels and servo motors controlling the claw's movement. This version demonstrated the core concept and validated our approach to autonomous litter collection.

V2

Tech Under 20

For Version 2, we moved away from the cardboard chassis and 3D-printed a professional chassis for the robot. We also upgraded the control system to an Arduino microcontroller. In this version, we focused primarily on improving movement and body design, with less emphasis on the claw mechanism. However, after finalizing the new chassis, we discovered that the old claw design was incompatible with the new body structure.

With the Tech Under 20 competition approaching in just one week, we didn't have time to design an elaborate claw arm like the previous version. Instead, we quickly 3D-printed a simplified, smaller claw that worked with the new body design. Beyond the technical development, we also had to create a comprehensive business pitch and proposal, as TU20 is an entrepreneurship competition focusing on real-world marketability and impact. This version, along with our business presentation, won us the Best Tech Team Award at the 2024 Tech Under 20 Cup.

Control System Code

The robot's control system was programmed in Arduino C/C++, focusing on motor control logic, sensor input handling, and reliable task sequencing. The system includes obstacle avoidance, object detection, precise movement control, and a multi-stage pickup/drop sequence. 

/*
 * A.L.T.R. - Adaptable Litter Taking Robot
 * Arduino Control System - Version 2
 * Developed for Tech Under 20 Competition 2024
 * 
 * Features:
 * - Dual motor control with speed modulation
 * - Ultrasonic obstacle detection
 * - Servo-controlled claw mechanism
 * - State machine for operation sequencing
 * - Error handling and safety checks
 */

#include <Servo.h>

// ========================================
// PIN DEFINITIONS
// ========================================
// Motor control pins (L298N driver)
const int LEFT_MOTOR_ENA = 5;      // Enable pin for left motor PWM
const int LEFT_MOTOR_IN1 = 6;      // Left motor direction control
const int LEFT_MOTOR_IN2 = 7;
const int RIGHT_MOTOR_ENB = 10;    // Enable pin for right motor PWM
const int RIGHT_MOTOR_IN3 = 8;     // Right motor direction control
const int RIGHT_MOTOR_IN4 = 9;

// Servo control pins
const int CLAW_SERVO_PIN = 3;      // Claw open/close servo
const int LIFT_SERVO_PIN = 11;     // Claw lift/lower servo

// Sensor pins
const int ULTRASONIC_TRIG = 12;    // Ultrasonic sensor trigger
const int ULTRASONIC_ECHO = 13;    // Ultrasonic sensor echo
const int IR_SENSOR_LEFT = A0;     // Left IR sensor for line following
const int IR_SENSOR_RIGHT = A1;    // Right IR sensor for line following

// Status LED
const int STATUS_LED = 2;

// ========================================
// SYSTEM CONSTANTS
// ========================================
// Motor speed parameters
const int BASE_SPEED = 180;        // Base forward speed (0-255)
const int TURN_SPEED = 140;        // Turning speed
const int SLOW_SPEED = 120;        // Slow movement for precision
const int SPEED_INCREMENT = 10;    // Speed adjustment increment

// Distance thresholds (cm)
const int OBSTACLE_DISTANCE = 25;  // Stop and avoid obstacle
const int OBJECT_DETECTION_DISTANCE = 15;  // Object within pickup range
const int MIN_DETECTION_DISTANCE = 3;      // Minimum valid distance
const int MAX_DETECTION_DISTANCE = 400;    // Maximum sensor range

// Servo positions (degrees)
const int CLAW_OPEN_POS = 30;      // Claw fully open
const int CLAW_CLOSE_POS = 120;    // Claw fully closed
const int CLAW_PARTIAL_CLOSE = 90; // Partial close for gripping
const int LIFT_UP_POS = 160;       // Claw lifted position
const int LIFT_DOWN_POS = 40;      // Claw lowered position
const int LIFT_MID_POS = 100;      // Intermediate position

// Timing constants (ms)
const int SCAN_DELAY = 300;        // Time to scan area
const int PICKUP_DELAY = 800;      // Delay between pickup steps
const int MOVEMENT_DELAY = 50;     // Delay between movement commands
const int SENSOR_READ_DELAY = 50;  // Delay between sensor reads

// Operation states
enum RobotState {
    STATE_IDLE,
    STATE_SCANNING,
    STATE_MOVING_FORWARD,
    STATE_DETECTING_OBJECT,
    STATE_PICKING_UP,
    STATE_MOVING_TO_DISPOSAL,
    STATE_DROPPING,
    STATE_AVOIDING_OBSTACLE,
    STATE_ERROR
};

// ========================================
// GLOBAL VARIABLES
// ========================================
Servo clawServo;
Servo liftServo;

RobotState currentState = STATE_IDLE;
RobotState previousState = STATE_IDLE;

unsigned long lastStateChange = 0;
unsigned long operationStartTime = 0;
int objectsCollected = 0;
int scanCount = 0;
bool systemInitialized = false;

// Movement tracking
int currentSpeed = BASE_SPEED;
bool isMoving = false;

// Sensor readings
long lastDistanceReading = 0;
long filteredDistance = 0;

// ========================================
// SETUP FUNCTION
// ========================================
void setup() {
    Serial.begin(9600);
    Serial.println("========================================");
    Serial.println("A.L.T.R. System Initializing...");
    Serial.println("Version 2.0 - Competition Build");
    Serial.println("========================================");
    
    // Initialize motor pins
    pinMode(LEFT_MOTOR_IN1, OUTPUT);
    pinMode(LEFT_MOTOR_IN2, OUTPUT);
    pinMode(RIGHT_MOTOR_IN3, OUTPUT);
    pinMode(RIGHT_MOTOR_IN4, OUTPUT);
    pinMode(LEFT_MOTOR_ENA, OUTPUT);
    pinMode(RIGHT_MOTOR_ENB, OUTPUT);
    
    // Initialize sensor pins
    pinMode(ULTRASONIC_TRIG, OUTPUT);
    pinMode(ULTRASONIC_ECHO, INPUT);
    pinMode(IR_SENSOR_LEFT, INPUT);
    pinMode(IR_SENSOR_RIGHT, INPUT);
    pinMode(STATUS_LED, OUTPUT);
    
    // Initialize servos
    clawServo.attach(CLAW_SERVO_PIN);
    liftServo.attach(LIFT_SERVO_PIN);
    
    // Calibrate servos to initial positions
    initializeServos();
    
    // Test sequence
    performSystemCheck();
    
    // Start in scanning state
    currentState = STATE_SCANNING;
    lastStateChange = millis();
    systemInitialized = true;
    
    Serial.println("System Ready. Beginning operation...");
    Serial.println("========================================");
    digitalWrite(STATUS_LED, HIGH);
}

// ========================================
// MAIN LOOP
// ========================================
void loop() {
    if (!systemInitialized) {
        errorState();
        return;
    }
    
    // State machine execution
    switch (currentState) {
        case STATE_SCANNING:
            performScan();
            break;
            
        case STATE_MOVING_FORWARD:
            moveForward();
            checkForObstacles();
            break;
            
        case STATE_DETECTING_OBJECT:
            if (detectObject()) {
                changeState(STATE_PICKING_UP);
            } else {
                changeState(STATE_SCANNING);
            }
            break;
            
        case STATE_PICKING_UP:
            executePickupSequence();
            break;
            
        case STATE_MOVING_TO_DISPOSAL:
            navigateToDisposal();
            break;
            
        case STATE_DROPPING:
            executeDropSequence();
            break;
            
        case STATE_AVOIDING_OBSTACLE:
            avoidObstacle();
            break;
            
        case STATE_ERROR:
            errorState();
            break;
            
        default:
            changeState(STATE_SCANNING);
            break;
    }
    
    // Update status LED
    updateStatusLED();
    
    delay(MOVEMENT_DELAY);
}

// ========================================
// STATE MANAGEMENT
// ========================================
void changeState(RobotState newState) {
    if (newState != currentState) {
        previousState = currentState;
        currentState = newState;
        lastStateChange = millis();
        Serial.print("State changed to: ");
        Serial.println(getStateName(newState));
    }
}

String getStateName(RobotState state) {
    switch (state) {
        case STATE_IDLE: return "IDLE";
        case STATE_SCANNING: return "SCANNING";
        case STATE_MOVING_FORWARD: return "MOVING_FORWARD";
        case STATE_DETECTING_OBJECT: return "DETECTING_OBJECT";
        case STATE_PICKING_UP: return "PICKING_UP";
        case STATE_MOVING_TO_DISPOSAL: return "MOVING_TO_DISPOSAL";
        case STATE_DROPPING: return "DROPPING";
        case STATE_AVOIDING_OBSTACLE: return "AVOIDING_OBSTACLE";
        case STATE_ERROR: return "ERROR";
        default: return "UNKNOWN";
    }
}

// ========================================
// SERVO CONTROL
// ========================================
void initializeServos() {
    Serial.println("Initializing servos...");
    clawServo.write(CLAW_OPEN_POS);
    delay(500);
    liftServo.write(LIFT_UP_POS);
    delay(500);
    Serial.println("Servos initialized");
}

void setClawPosition(int position, int speed = 15) {
    int currentPos = clawServo.read();
    if (position > currentPos) {
        for (int pos = currentPos; pos <= position; pos++) {
            clawServo.write(pos);
            delay(speed);
        }
    } else {
        for (int pos = currentPos; pos >= position; pos--) {
            clawServo.write(pos);
            delay(speed);
        }
    }
}

void setLiftPosition(int position, int speed = 15) {
    int currentPos = liftServo.read();
    if (position > currentPos) {
        for (int pos = currentPos; pos <= position; pos++) {
            liftServo.write(pos);
            delay(speed);
        }
    } else {
        for (int pos = currentPos; pos >= position; pos--) {
            liftServo.write(pos);
            delay(speed);
        }
    }
}

// ========================================
// MOTOR CONTROL
// ========================================
void setMotorSpeed(int leftSpeed, int rightSpeed) {
    // Constrain speeds to valid PWM range
    leftSpeed = constrain(leftSpeed, -255, 255);
    rightSpeed = constrain(rightSpeed, -255, 255);
    
    // Left motor control
    if (leftSpeed > 0) {
        digitalWrite(LEFT_MOTOR_IN1, HIGH);
        digitalWrite(LEFT_MOTOR_IN2, LOW);
        analogWrite(LEFT_MOTOR_ENA, abs(leftSpeed));
    } else if (leftSpeed < 0) {
        digitalWrite(LEFT_MOTOR_IN1, LOW);
        digitalWrite(LEFT_MOTOR_IN2, HIGH);
        analogWrite(LEFT_MOTOR_ENA, abs(leftSpeed));
    } else {
        digitalWrite(LEFT_MOTOR_IN1, LOW);
        digitalWrite(LEFT_MOTOR_IN2, LOW);
        analogWrite(LEFT_MOTOR_ENA, 0);
    }
    
    // Right motor control
    if (rightSpeed > 0) {
        digitalWrite(RIGHT_MOTOR_IN3, HIGH);
        digitalWrite(RIGHT_MOTOR_IN4, LOW);
        analogWrite(RIGHT_MOTOR_ENB, abs(rightSpeed));
    } else if (rightSpeed < 0) {
        digitalWrite(RIGHT_MOTOR_IN3, LOW);
        digitalWrite(RIGHT_MOTOR_IN4, HIGH);
        analogWrite(RIGHT_MOTOR_ENB, abs(rightSpeed));
    } else {
        digitalWrite(RIGHT_MOTOR_IN3, LOW);
        digitalWrite(RIGHT_MOTOR_IN4, LOW);
        analogWrite(RIGHT_MOTOR_ENB, 0);
    }
    
    isMoving = (leftSpeed != 0 || rightSpeed != 0);
}

void moveForward() {
    setMotorSpeed(currentSpeed, currentSpeed);
}

void moveBackward(int duration = 1000) {
    setMotorSpeed(-SLOW_SPEED, -SLOW_SPEED);
    delay(duration);
    stopMotors();
}

void turnLeft(int angle = 45) {
    // Approximate turning based on angle
    int turnTime = map(angle, 0, 360, 0, 800);
    setMotorSpeed(-TURN_SPEED, TURN_SPEED);
    delay(turnTime);
    stopMotors();
}

void turnRight(int angle = 45) {
    int turnTime = map(angle, 0, 360, 0, 800);
    setMotorSpeed(TURN_SPEED, -TURN_SPEED);
    delay(turnTime);
    stopMotors();
}

void stopMotors() {
    setMotorSpeed(0, 0);
    isMoving = false;
}

void pivotTurnLeft() {
    setMotorSpeed(-TURN_SPEED, TURN_SPEED);
}

void pivotTurnRight() {
    setMotorSpeed(TURN_SPEED, -TURN_SPEED);
}

// ========================================
// ULTRASONIC SENSOR
// ========================================
long readUltrasonicDistance() {
    // Clear trigger pin
    digitalWrite(ULTRASONIC_TRIG, LOW);
    delayMicroseconds(2);
    
    // Send 10us pulse to trigger
    digitalWrite(ULTRASONIC_TRIG, HIGH);
    delayMicroseconds(10);
    digitalWrite(ULTRASONIC_TRIG, LOW);
    
    // Read echo pulse duration
    long duration = pulseIn(ULTRASONIC_ECHO, HIGH, 30000); // 30ms timeout
    
    // Calculate distance in cm
    long distance = (duration / 2) / 29.1;
    
    // Validate reading
    if (distance < MIN_DETECTION_DISTANCE || distance > MAX_DETECTION_DISTANCE) {
        return -1; // Invalid reading
    }
    
    return distance;
}

long getFilteredDistance(int samples = 5) {
    long sum = 0;
    int validReadings = 0;
    
    for (int i = 0; i < samples; i++) {
        long reading = readUltrasonicDistance();
        if (reading > 0) {
            sum += reading;
            validReadings++;
        }
        delay(20);
    }
    
    if (validReadings > 0) {
        filteredDistance = sum / validReadings;
        return filteredDistance;
    }
    
    return -1; // No valid readings
}

bool detectObject() {
    long distance = getFilteredDistance(3);
    if (distance > 0) {
        lastDistanceReading = distance;
        return (distance >= MIN_DETECTION_DISTANCE && distance <= OBJECT_DETECTION_DISTANCE);
    }
    return false;
}

bool checkForObstacles() {
    long distance = readUltrasonicDistance();
    if (distance > 0 && distance < OBSTACLE_DISTANCE) {
        changeState(STATE_AVOIDING_OBSTACLE);
        return true;
    }
    return false;
}

// ========================================
// OPERATION SEQUENCES
// ========================================
void performScan() {
    Serial.println("Scanning area for objects...");
    scanCount++;
    
    // Rotate left to scan
    pivotTurnLeft();
    delay(SCAN_DELAY);
    stopMotors();
    delay(200);
    
    // Check for objects
    if (detectObject()) {
        Serial.println("Object detected!");
        changeState(STATE_DETECTING_OBJECT);
    } else {
        // Rotate back
        pivotTurnRight();
        delay(SCAN_DELAY);
        stopMotors();
        delay(200);
        
        // If no object found after multiple scans, move forward
        if (scanCount > 3) {
            scanCount = 0;
            changeState(STATE_MOVING_FORWARD);
        }
    }
}

void executePickupSequence() {
    Serial.println("=== Pickup Sequence Initiated ===");
    stopMotors();
    delay(300);
    
    // Step 1: Position robot closer to object
    Serial.println("Positioning for pickup...");
    long distance = getFilteredDistance();
    if (distance > OBJECT_DETECTION_DISTANCE) {
        moveForward();
        delay(map(distance, OBJECT_DETECTION_DISTANCE, OBSTACLE_DISTANCE, 200, 800));
        stopMotors();
    }
    
    // Step 2: Lower the claw
    Serial.println("Lowering claw...");
    setLiftPosition(LIFT_DOWN_POS, 12);
    delay(PICKUP_DELAY);
    
    // Step 3: Close claw gradually for better grip
    Serial.println("Closing claw...");
    setClawPosition(CLAW_PARTIAL_CLOSE, 8);
    delay(300);
    setClawPosition(CLAW_CLOSE_POS, 10);
    delay(500);
    
    // Step 4: Verify grip (optional - could use sensor)
    delay(200);
    
    // Step 5: Lift the claw
    Serial.println("Lifting object...");
    setLiftPosition(LIFT_UP_POS, 12);
    delay(PICKUP_DELAY);
    
    objectsCollected++;
    Serial.print("Object picked up successfully! Total: ");
    Serial.println(objectsCollected);
    Serial.println("========================================");
    
    changeState(STATE_MOVING_TO_DISPOSAL);
}

void navigateToDisposal() {
    Serial.println("Navigating to disposal area...");
    
    // Turn around
    turnRight(180);
    delay(300);
    
    // Move backward to disposal area
    moveBackward(1500);
    
    // Small adjustment
    turnLeft(30);
    delay(200);
    
    changeState(STATE_DROPPING);
}

void executeDropSequence() {
    Serial.println("=== Drop Sequence Initiated ===");
    stopMotors();
    delay(300);
    
    // Step 1: Lower the claw
    Serial.println("Lowering claw...");
    setLiftPosition(LIFT_DOWN_POS, 12);
    delay(PICKUP_DELAY);
    
    // Step 2: Open the claw
    Serial.println("Releasing object...");
    setClawPosition(CLAW_OPEN_POS, 8);
    delay(600);
    
    // Step 3: Lift the claw
    Serial.println("Raising claw...");
    setLiftPosition(LIFT_UP_POS, 12);
    delay(PICKUP_DELAY);
    
    Serial.println("Object dropped successfully!");
    Serial.println("========================================");
    
    // Return to scanning
    changeState(STATE_SCANNING);
    scanCount = 0;
}

void avoidObstacle() {
    Serial.println("Obstacle detected! Avoiding...");
    stopMotors();
    delay(300);
    
    // Move backward slightly
    moveBackward(500);
    
    // Turn right to avoid
    turnRight(90);
    delay(300);
    
    // Continue forward
    changeState(STATE_MOVING_FORWARD);
}

// ========================================
// SYSTEM FUNCTIONS
// ========================================
void performSystemCheck() {
    Serial.println("Performing system check...");
    
    // Test motors
    Serial.println("Testing motors...");
    setMotorSpeed(100, 100);
    delay(200);
    stopMotors();
    delay(200);
    setMotorSpeed(-100, -100);
    delay(200);
    stopMotors();
    
    // Test servos
    Serial.println("Testing servos...");
    setLiftPosition(LIFT_MID_POS);
    delay(300);
    setClawPosition(CLAW_PARTIAL_CLOSE);
    delay(300);
    initializeServos();
    
    // Test sensors
    Serial.println("Testing sensors...");
    for (int i = 0; i < 3; i++) {
        long dist = readUltrasonicDistance();
        Serial.print("Distance reading: ");
        Serial.println(dist);
        delay(200);
    }
    
    Serial.println("System check complete!");
}

void updateStatusLED() {
    // Blink pattern based on state
    static unsigned long lastBlink = 0;
    static bool ledState = false;
    
    unsigned long blinkInterval = 500;
    if (currentState == STATE_ERROR) {
        blinkInterval = 100; // Fast blink for error
    } else if (currentState == STATE_PICKING_UP || currentState == STATE_DROPPING) {
        blinkInterval = 200; // Medium blink during operations
    }
    
    if (millis() - lastBlink > blinkInterval) {
        ledState = !ledState;
        digitalWrite(STATUS_LED, ledState);
        lastBlink = millis();
    }
}

void errorState() {
    Serial.println("ERROR: System malfunction!");
    stopMotors();
    digitalWrite(STATUS_LED, HIGH);
    delay(100);
    digitalWrite(STATUS_LED, LOW);
    
    // Attempt recovery after delay
    if (millis() - lastStateChange > 5000) {
        Serial.println("Attempting system recovery...");
        performSystemCheck();
        changeState(STATE_SCANNING);
    }
}
